Class: Admin

Admin(nome, senha)

Classe que representa um administrador, que herda da classe Usuario. O administrador pode realizar operações específicas, como trocar senhas de usuários.

trocaSenha()

Permite que o administrador troque a senha de um usuário. O administrador pode buscar o usuário por nome ou ID e confirmar a troca de senha. Se o usuário for encontrado, a senha será alterada conforme a confirmação do administrador.
