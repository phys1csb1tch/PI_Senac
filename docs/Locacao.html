Classe para gerenciar locações

Constructor

new Locacao(idLivro, idUsuario, dataLocacao, dataDevolucao, dataDevolvida)

Representa uma locação de livro.

Exemplo de uso:

const locacao = new Locacao(1, 101, '2024-12-01', '2024-12-15', '2024-12-10');

O ID do usuário que está fazendo a locação.

A data em que o livro foi locado.
dataDevolucao string

A data em que o livro deve ser devolvido.
dataDevolvida string

A data em que o livro foi realmente devolvido.
